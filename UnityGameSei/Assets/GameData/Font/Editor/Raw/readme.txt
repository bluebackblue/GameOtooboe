＋＋＋フォントの使い方＋＋＋
解凍したazuki.ttfをコントロールパネルのフォントフォルダに入れてください。
尚、MacOSX未満には対応しておりません。ご了承ください。

＋＋＋利用規約＋＋＋
http://azukifont.com/kiyaku.html
をご覧下さい。


＋＋罫線、特殊文字、絵文字の出し方＋＋
「けいせん」を変換すると出ます。
